# RegentFemCards / 储君娘化卡图

I made this mod to add Regent Fem-themed card art and enrich the Slay the Spire 2 experience.  
我制作了这个 Mod，用来加入储君主题的娘化卡图，丰富《杀戮尖塔2》的游戏体验。

---

## Requirements / 运行需求

Before using this mod, make sure you have the following installed:  
在使用本 Mod 之前，请确保你已经安装以下内容：

- **Slay the Spire 2**

---

## Installation / 安装方法

### English

1. Download the mod files.
2. Extract the downloaded archive.
3. Put the mod folder into the `mods` folder under your Slay the Spire 2 game directory.
4. If the `mods` folder does not exist, create one manually.
5. Launch the game.

### 中文

1. 下载本 Mod 文件。
2. 解压下载得到的压缩包。
3. 将解压后的 Mod 文件夹 文件放入《杀戮尖塔2》游戏根目录下的 `mods` 文件夹中。
4. 如果没有 `mods` 文件夹，就手动新建一个。
5. 启动游戏。

## Save File Notice / 存档说明

### English

If you have never installed mods before, please note that Slay the Spire 2 separates save data between **modded** and **unmodded** modes.

If you want to keep your original saves, go to:

`C:\Users\<YourUserName>\AppData\Roaming\SlayTheSpire2\steam\<YourSteamID>`

Then copy your `profile_0`, `profile_1`, `profile_2` or other profile files into the corresponding modded save location if needed.

If you the `modded` folder does not exist, lanching the game with mod enabled will automatically create it.

If you cannot see the `AppData` folder, enable **Hidden items** in File Explorer.

### 中文

如果你以前从来没有安装过 Mod，需要注意：《杀戮尖塔2》的存档会区分 **已安装 Mod** 和 **未安装 Mod** 两种模式。

如果你想保留原来的存档，请前往：

`C:\Users\你的用户名\AppData\Roaming\SlayTheSpire2\steam\你的SteamID`

然后根据需要，将其中的 `profile_0`、`profile_1`、`profile_2` 等存档文件复制到 modded 存档位置中。

如果没有 `modded` 文件夹，就先安装mod运行一次游戏。

如果你看不到 `AppData` 文件夹，请在文件资源管理器中勾选 **显示隐藏的项目**。
